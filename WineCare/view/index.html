WineCare

É um projeto de monitoramento para Barris de Vinhos de alto padrão.

Sobre

Sabemos a importância da refrigeração e armazenamento de vinhos para que os mesmos cheguem em excelentes condições para consumo. Por isso, a WineCare é um Sistema que busca fornecer uma solução tecnológica para grandes vinícolas brasileiras para que tenham melhor controle da temperatura dos barris de vinho.

Serviços

Sistema Web

Sistema Web totalmente responsivo e acessívelutilizado para Gerenciar seus Barris.

Sensores

Sensores para captação de dados de temperatura e umidade.

Controle de Dados

Dashboard para visualizar os dados coletados pelos Sensores.

Planos

Onde queremos chegar?

A nossa missão é auxiliar os produtores a reduzir em até 50% as perdas de vinhos nobres durante o processo de maturação.

Com nossa aplicação web, nossos clientes podem monitorar em tempo real a temperatura e a umidade dos vinhos armazenados, acessando facilmente os dados capturados por nossos sensores.

Fornecemos alertas que permitem aos produtores agir prontamente, mantendo a qualidade do vinho.
