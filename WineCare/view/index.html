WineCare

Monitoramento de Barris de Vinho.

Sobre

Sabemos a importância da refrigeração e armazenamento de vinhos para que os mesmos cheguem em excelentes condições para consumo.

Por isso, a WineCare é um Sistema que busca fornecer uma solução em IOT para grandes vinícolas brasileiras para que tenham melhor controle da temperatura dos barris de vinho

Serviços

Sistema Web

Sistema Web totalmente responsivo e acessível, utilizado para Gerenciar seus Barris.

Sensores

Sensores para captação de dados de temperatura e umidade.

Controle de Dados

Dashboard para visualizar os dados coletados pelos Sensores.
